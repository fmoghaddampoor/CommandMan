Project: CommandMan.UI

The UI is an Angular 18+ application rendered inside WebView2.

Directory Structure

CommandMan.UI/src/app/
├── app.component.ts/html/scss  # Main application component
├── app.config.ts               # Angular configuration
├── services/
│   ├── bridge.service.ts       # WebView2 communication
│   └── progress.service.ts     # Progress state management
├── file-pane/                  # File listing component
├── breadcrumb/                 # Path navigation component
├── progress-overlay/           # Progress modal component
├── input-dialog/               # Text input dialog
└── about-dialog/               # About application dialog

AppComponent

The root component that orchestrates the dual-pane file manager layout.

Components

FilePaneComponent

Input/Output Type Description
@Input() paneId 'left' | 'right' Identifies which pane
items FileSystemItem[] List of files/folders
currentPath string Current directory path
selectedItems Set<string> Selected item paths

BreadcrumbComponent

Displays and navigates the current path hierarchy.

ProgressOverlayComponent

Shows operation progress with percentage and current file.

InputDialogComponent

Generic dialog for text input (create folder, rename).

State Management

The UI uses RxJS BehaviorSubjects for reactive state management.

Pane State

interface PaneState {
  items: FileSystemItem[];    // Current directory contents
  currentPath: string;        // Current path
  focusItem?: string;         // Item to focus after operation
}

// Observable streams
leftPane$: BehaviorSubject<PaneState>
rightPane$: BehaviorSubject<PaneState>
drives$: BehaviorSubject<DriveItem[]>

Keyboard Shortcuts

Key Action
F3 View selected file
F4 Edit selected file
F5 Copy selected to other pane
F6 Move selected to other pane
F7 Create new directory
F8 / Delete Delete selected items
Enter Open selected item
Tab Switch active pane

Styling

The UI uses a dark Catppuccin-inspired theme with SCSS.

Color Palette

$background: #11111b;      // Base background
$surface: #1e1e2e;         // Panel background
$text: #cdd6f4;            // Primary text
$accent: #89b4fa;          // Accent color (blue)
$selected: #313244;        // Selected item background

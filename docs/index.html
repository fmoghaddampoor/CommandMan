CommandMan - File Manager

Version: 1.0.0 | Platform: Windows

Welcome to CommandMan

CommandMan is a modern dual-pane file manager built with:

  • WPF Shell - .NET 10.0 Windows Presentation Foundation
  • Angular UI - Angular 18+ frontend rendered in WebView2
  • WebView2 Bridge - Bidirectional communication layer

Quick Start

Running in Development

# Terminal 1: Start Angular dev server
cd CommandMan.UI
npm install
npm run start

# Terminal 2: Run WPF application
cd CommandMan.Shell
dotnet run

Building for Production

# Run the build script
.\build.ps1 -Clean
Note: CommandMan uses WebView2 runtime which is included with Windows 11 and available as a separate download for Windows 10.
